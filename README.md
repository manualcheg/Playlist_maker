# Playlist_maker

Это приложение для создания плейлистов с функцией прослушивания отрывков треков и возможностью добавления треков в избранное.
Создано в рамках обучения на курсе Android-разработчик от Яндекс Практикум. 

Использовано:
- Kotlin 
- MVVM 
- Retrofit
- Koin
- Room
- Clean Architecture
- Jetpack navigation
- Api iTunes
